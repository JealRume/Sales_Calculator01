# Sales Calculatorv2

## Description

This project is a simple Python program for calculating a sales summary.  
The script is designed to obtain customer data and calculate the total cost of a purchase.  
The system is able to apply a discount when the customer is a VIP.

## How it works

1. The program imports two functions from external modules.
2. The function `data_entry` collects the customer information.
3. The program receives the client name, quantity, unit price, and VIP status.
4. The function `calculate_totals` calculates the subtotal, discount, and total.
5. The program checks if the client is VIP.
6. The system prints a summary of the sale including the total payment.
